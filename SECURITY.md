# Security Policy

## Reporting a Vulnerability

**Please do not open public issues for security bugs.**

If you discover a vulnerability in VAULTX, please report it privately. You can submit details to:

- **Security Team Email**: security@vaultx.io

Please include:
- A description of the issue.
- Steps to reproduce or a Proof of Concept (PoC).
- Potential impact of the vulnerability.

We will acknowledge receipt of your report within 48 hours and work with you to coordinate a secure release patch. Thank you for helping keep VAULTX secure!
